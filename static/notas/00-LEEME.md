# Notas del proyecto UAdeO Tienda Lince

Esta carpeta contiene explicaciones en español, escritas para entender
**cómo funciona** el proyecto sin tener que leer todo el código.

No se sube a producción ni se ejecuta nada de aquí. Es solo material
de estudio para el equipo.

## Índice

| Archivo | Tema |
|---|---|
| `01-arquitectura-general.md` | Cómo se comunican el frontend (HTML/CSS/JS) y el backend (Django). |
| `02-navbar-dinamica.md` | Cómo el JavaScript inyecta la barra de navegación en cada página. |
| `03-flujo-de-una-pagina.md` | Qué pasa paso a paso cuando el usuario abre una URL. |
| `04-autenticacion.md` | Registro, login, logout, CSRF y sesiones explicado paso a paso. |
| `05-catalogo-y-fetch.md` | Cómo se carga la lista de productos, los filtros y el buscador. |
| `06-carrito-localstorage.md` | Cómo funciona el carrito guardado en el navegador (sin tocar el backend). |
| `07-pedidos-y-confirmacion.md` | Confirmar apartado, crear pedido en la BD y ver el historial. |
| `08-checklist-entrega-y-pruebas.md` | Lista final para probar todo antes de entregar el proyecto. |

> Consejo: léelos en orden. Cada uno se apoya en el anterior.
